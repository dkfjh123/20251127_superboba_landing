## 1. Overview

### 1.1 Design Goals

1. 스크롤 = 설득 여정: 멈추게 → 공감 → 믿게 → 사게 순서를 시각 리듬(화이트→다크→크림)으로 체화
2. 히어로 3초 룰: 리빌 이펙트 + 타겟 지목 + 신뢰 배지로 이탈 차단
3. 어느 지점에서든 문의까지 1클릭 (스티키 헤더 CTA)
4. 단일 파일·경량 유지 (외부 의존성: 폰트 CDN + R2 이미지뿐)

### 1.2 Design Principles

- 섹션 하나당 메시지 하나 (상세페이지 풀프로세스 원칙)
- 기능이 아닌 결과로 말한다 (객단가·차별화·운영부담)
- 🟡 미확정 사실은 페이지에 싣지 않는다 — "상담 시 안내"로 처리
- 레드는 포인트로만 — CTA·강조에 집중

## 2. Architecture Options

### 2.0 Architecture Comparison

| Criteria | A: 기존 파일 재배치 | B: 3파일 분리 | **C: 단일파일 재작성** |
|----------|:-:|:-:|:-:|
| Approach | 섹션 추가·재배열 | html/css/js 분리 | 새로 쓰되 검증 자산 이식 |
| New Files | 0 | 3 | 1 (index.html 교체) |
| Complexity | Low | Medium | Medium |
| Maintainability | Low | High | Medium-High |
| Effort | Low | High | Medium |
| Risk | 레이아웃 한계·리빌 억지 이식 | 배포 복잡 | Low |

**Selected**: **Option C** — 배포 단순함(파일 1개) 유지 + 새 설득 구조 자유도. 이식 자산: 레드 토큰, formsubmit 폼(+접수 알림 JS), FAQ 아코디언 JS, 모바일 캐러셀 패턴.

### 5.2 S1 Hero 상세 (가장 중요)

```
┌─────────────────────────────────────────┐
│ [배지] 카페 사장님을 위한 B2B 전용 공급   │
│                                         │
│ H1: 매일 삶지 않아도,                    │
│     슈퍼보바 직영점의 그 맛 그대로.       │  ← "그 맛 그대로" 레드 그라디언트
│                                         │
│ 서브: 직영 매장에서 검증된 수제 생보바를,  │
│       전국 카페가 받아 쓰는 전용상품으로. │
│                                         │
│ [도입 상담 신청(레드)] [전용상품 보기]     │
│                                         │
│ ┌───────────────────────────────┐       │
│ │   리빌 이미지 스테이지          │       │
│ │   base: 완성 흑당 생보바 음료   │       │
│ │   over: 생가루 반죽/도우 성형    │       │
│ │   (스크롤 ↓ → 반죽이 걷히며     │       │
│ │    완성 음료가 드러남)          │       │
│ └───────────────────────────────┘       │
│ [신뢰 배지 스트립]                        │
│ 15년 F&B R&D · 갤러리아 입점 · 리뷰 187  │
└─────────────────────────────────────────┘
```

**리빌 이펙트 기술 스펙**
- 같은 구도 이미지 2장 절대 겹침(position:absolute), 위 레이어(반죽)를 `clip-path: inset(0 0 X% 0)`으로 스크롤 진행도에 따라 걷어냄
- 구동: `scroll` 이벤트 + `requestAnimationFrame`, 히어로 구간 스크롤 진행도 0→1 매핑 (라이브러리 無)
- `prefers-reduced-motion` → 크로스페이드(opacity) 폴백
- 모바일: 동일 로직 유지하되 스테이지 높이 축소 (60vh)
- 이미지: 힉스필드 페어 생성 — "같은 구도·같은 조명, 반죽 상태 vs 완성 음료" (실사진 R2 자산을 스타일 참조로). 생성 전까지 기존 R2 음료 사진 + 반투명 크림 오버레이 플레이스홀더
- 스크롤 유도: 하단 중앙 "스크롤해서 확인하세요 ↓" 마이크로카피

  - ① 국내 차(茶) 시장: **1.10조 → 1.58조 (+44%)** (2020→2024, 유로모니터·통계청) — "커피 포화 속, 차만 성장"
