# 2026-07-16 작업 히스토리

> 주제: 사이트 법적·기술 필수 항목 마감 (개인정보 동의·sitemap) + 비즈채널 승인 확인 → 파워링크 준비 완료

## 오늘 한 일

### 1. 개인정보 수집·이용 동의 추가 (커밋 952ccc4) — 법적 필수
- 배경: 문의 폼이 매장명·성함·연락처를 수집하는데 동의 절차가 없었음. 메타 픽셀(전날 설치, 커밋 4b0af60)·GA4가 도는 상태라 데이터 수집 고지도 필요.
- 폼에 **필수 동의 체크박스** 추가 — 미체크 시 전송 불가(브라우저 기본 검증). 체크 여부는 문의 메일에 "개인정보동의: 동의함"으로 기록되어 증빙이 남음.
- "자세히 보기"(details 접이식)로 전문 고지: 수집 항목(매장명·성함·연락처·문의내용) / 이용 목적(도입 상담) / **보유 기간: 상담 완료 후 6개월** / 거부 권리 / **GA·Meta Pixel 사용 및 쿠키 수집 고지**
- 전송 JS·Lead 이벤트는 미변경 → 광고 성과 측정에 영향 없음. 배포 후 실사이트에서 노출 확인 완료.

### 2. robots.txt + sitemap.xml 추가 (커밋 952ccc4)
- robots.txt: 전체 허용 + Sitemap 지정 / sitemap.xml: 단일 URL(`/`) 등록
- ⚠ 기록: **Cloudflare가 robots.txt 앞부분에 AI 크롤러 차단 블록(ClaudeBot·GPTBot 등)을 자동 주입**함 — 우리가 쓴 `Allow: /` + Sitemap은 뒤에 정상 서빙되고, 네이버·구글 검색 수집에는 영향 없음.
- 파비콘은 이미 설치돼 있었음(index.html `superboba-symbol.png`) — 추가 작업 불필요.

### 3. 네이버 서치어드바이저 — 사이트맵 제출 완료
- 요청 > 사이트맵 제출에서 `sitemap.xml` 입력·제출 (사장님 직접). 수집·색인은 며칠~수 주 소요, 추가 조치 불필요.
- 서치어드바이저는 승인 개념 없음 — 이후 확인은 "리포트 > 수집 현황" 또는 네이버에서 `site:superboba.co.kr` 검색.

### 4. 네이버 검색광고 — 비즈채널 심사 통과 ✅
- "슈퍼보바 랜딩" 웹사이트 채널: **PC 노출가능 · 모바일 노출가능** (심사 1영업일 만에 승인)
- 채널 상세: 파비콘 수집 성공 / 전환 추적 관리 "사용 안 함"(선택 항목, 추후)
- → 파워링크 캠페인 생성 가능 상태가 됨.

### 5. 파워링크 캠페인 준비물 확정 (다음 세션에 바로 사용)
- 캠페인: 이름 `슈퍼보바_파워링크` / 하루예산 20,000원
- 광고그룹 3개 + 키워드 후보:
  - ① 펄 납품: 타피오카펄, 타피오카펄 도매, 버블티 납품, 버블티 재료, 흑당펄, 생보바
  - ② 카페 신메뉴: 카페 신메뉴, 카페 신메뉴 추천, 카페 디저트 납품, 카페 음료 재료
  - ③ 슈퍼호두: 호두과자 창업, 호두과자 납품, 카페 디저트 도매, 슈퍼보바
- 광고 소재(제목 15자/설명 45자 규격, 브랜드 규칙 준수 확인됨):
  - ① `직영점 검증 생보바 납품` / `매일 반죽해 삶는 수제 생보바, 직영점에서 검증한 맛을 카페에 공급합니다.`
  - ② `카페 신메뉴, 슈퍼보바` / `반죽하고 삶는 일은 슈퍼보바가 합니다. 도입 상담은 대표가 직접 안내합니다.`
  - ③ `슈퍼호두 B2B 공급` / `일반 호두과자 3배 크기, 직영점 검증 디저트를 카페 전용상품으로 공급합니다.`
